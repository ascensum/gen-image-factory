# About

## Gen Image Factory

**Gen Image Factory** is a professional AI-powered image generation and processing tool designed to streamline your creative workflow.

### Made by Shiftline Tools

Gen Image Factory is developed by an individual developer under the **Shiftline Tools** brand, a software development studio focused on creating tools that make your workflow easier.

## What is Gen Image Factory?

Gen Image Factory helps you:

- **Generate Images**: Create high-quality images using AI-powered generation
- **Process Images**: Automate image processing tasks including background removal
- **Quality Control**: Use AI to automatically review and filter generated images
- **Export & Organize**: Export results with metadata in multiple formats (ZIP, Excel, CSV)

## Features

- **AI-Powered Generation**: Leverage advanced AI models for image creation
- **Batch Processing**: Process multiple images efficiently
- **Quality Checks**: Automated quality assessment using AI
- **Metadata Generation**: Automatic metadata creation for generated images
- **Cross-Platform**: Available for Windows, macOS, and Linux
- **Microsoft Store**: Available on Windows through Microsoft Store for easy installation and updates

## Distribution

Gen Image Factory is distributed through:

- **Microsoft Store** (Windows) - Recommended for Windows users
- **GitHub Releases** - For all platforms, including advanced installation options
