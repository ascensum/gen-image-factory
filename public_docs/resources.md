**API Keys and Costs**: Gen Image Factory requires you to provide your own API keys for OpenAI and Runware services. These keys are stored securely using your operating system's credential manager. You are solely responsible for:

- Obtaining valid API keys from each service provider
- Any costs or charges incurred through third-party API usage
- Complying with each service provider's terms of service
- Managing your API usage and billing with each provider

**Service Availability**: Shiftline Tools is not responsible for third-party API availability, data handling, or service interruptions. For issues related to third-party services, please contact the respective service provider directly.

For more detailed information about API usage terms, please see our [Terms of Service](/legal/terms-of-service#api-usage-terms).
